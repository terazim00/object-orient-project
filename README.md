- ## 📌 참고사항
이 코드는 **구조체(Structure)만 구현**한 것으로:
- 모든 함수는 `pass`로 선언되어 있습니다
- 실제 구현은 각 함수 내부에 주석으로 설명되어 있습니다
- 추후 개발 시 각 함수의 로직을 구현하면 됩니다
